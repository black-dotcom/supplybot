# TG供需机器人

🔥TG供需机器人

❓介绍：

客户可自助发布广告,可设置收费,可设置是否审核。审核通过自动发布到指定频道/群组.

📣特点：

✅  内置USDT收款 无需第三方支付接口

✅ Python语言编写运行高效配置简单

✅ 公群 流量 担保 必备

💲更多功能可定制，演示查看 @deepluobo_bot
